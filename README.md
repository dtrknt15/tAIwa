# 🌐 tAIwa - マルチペルソナ チャットルーム

**tAIwa** は、複数のAIエージェント（ペルソナ）と人間（あなた）が、一つのチャットルームに集まって設定したテーマ（アジェンダ）について自由に対話・討論を行うことができる Gradio ベースの対話システムです。

**HuggingFace Spaceで動くバージョン**
<https://huggingface.co/spaces/dtrknt/tAIwa>

---

## ✨ 主な機能

* 👥 **マルチペルソナ対話**: 最大4つのペルソナを設定し、AI同士または人間を交えたグループディスカッションが行えます。
* ⚙️ **OpenAI互換API対応**:
  * OpenAI API はもちろん、ローカルで動作する API サーバー（LM Studio、Ollama、vLLM など）や Hugging Face Serverless Inference 等の様々な OpenAI 互換のエンドポイントに接続可能です。
  * エンドポイントから利用可能なモデル一覧を自動取得する機能も備えています。
* 🌐 **多言語対応 (日本語/英語)**: ブラウザの言語設定を自動的に検出し、UIを最適な言語（日本語または英語）で表示します。
* 🎨 **プレミアムUIデザイン**:
  * ライトモードとダークモードの両方に最適化された美しいグラスモーフィズムデザイン。
  * 発言者（ペルソナ）ごとに見やすく色分けされたチャットバブル表示。
  * レスポンシブで快適な操作感。

## 🚀 セットアップと起動方法

### 1. 必要なライブラリのインストール

Python 3.8以上が必要です。以下のコマンドで必要な依存関係をインストールします：

```bash
pip install gradio openai
```

### 2. 環境変数の設定 (任意)

あらかじめ API キーを設定しておくと、起動時に入力する手間を省くことができます。

**Windows (PowerShell):**

```powershell
$env:OPENAI_API_KEY="your-api-key"
```

**macOS / Linux:**

```bash
export OPENAI_API_KEY="your-api-key"
```

### 3. アプリケーションの起動

プロジェクトのルートディレクトリで以下のコマンドを実行します：

```bash
python main.py
```

起動後、コンソールに表示されるURL（デフォルトでは `http://127.0.0.1:7860`）をブラウザで開いてください。

## 💡 使い方

1. **API設定 (画面左上)**:
    * APIキー、ベースURL、モデル名を設定します。ローカルサーバーを使用する場合は、ベースURLにローカルアドレス（例: `http://localhost:1234/v1`）を指定してください。
    * 🔄 ボタンをクリックすると、接続先APIから利用可能なモデルリストを読み込んで選択することができます。
2. **ペルソナ設定 (画面左下)**:
    * ディスカッションに参加させる人数（2〜4人）を設定します。
    * 各ペルソナの「名前」「AIか人間（Checkbox）」「システムプロンプト」を設定します。
3. **アジェンダ設定とディスカッション開始 (画面右)**:
    * ディスカッションのテーマ（アジェンダ）を入力します。
    * **🚀 開始する** ボタンをクリックしてチャットを開始します。
4. **会話の進行**:
    * AIのターンの場合は、**➡️ 次のターン** ボタンを押すとAIの返答が生成されます。
    * あなたのターンの場合は、画面下部に返答入力欄が表示されるので、メッセージを入力して **送信** してください。
    * 途中でやり直したい場合は **🔄 リセット** ボタンで初期状態に戻せます。
